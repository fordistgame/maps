启动说明

步骤一：先关闭杀毒（少数电脑不需要关闭，可忽略此步骤）

如果报毒属于误报，修改器基本都会报毒

如果电脑有windows defender也需要关闭

步骤二：解压文件

步骤三：右键管理员启动程序，输入卡密登录，等待提示激活成功

步骤四：打开游戏，游戏菜单界面按住Ctl+Insert

如果是笔记本电脑请按住Ctl+Fn+Insert

如果是笔记本电脑外接了键盘，请用笔记本电脑自身键盘

如果以上方法都没用，就按Win+R

如果依然没用请联系售后

菜单操作说明(部分游戏修改器没有菜单，或外置菜单，可忽略此说明)

[HOME]键显示或隐藏菜单

[↑][↓]方向键切换菜单栏目

[←][→]方向键修改栏目内容

[PgDn]进入下级菜单

[Backspace]撤销键返回上级菜单

Startup Instructions

Step 1: Disable antivirus software (some computers may not require this step, which can be ignored)

If the antivirus software reports a false positive, the cheat tool will likely trigger it

If the computer has Windows Defender, it needs to be disabled as well.

Step 2: Extract the files.

Step 3: Right-click to run the program as an administrator, enter the activation code, and wait for the prompt to indicate successful activation.

Step 4: Open the game and press Ctl+Insert on the game menu screen

If it's a laptop, please press Ctl+Fn+Insert

If a keyboard is connected to the laptop, use the laptop's own keyboard instead

If the above methods do not work, press Win+R

If none of the above methods work, please contact customer service

Menu Operation Instructions (some game modifiers may not have a menu, or have an external menu, this instruction can be ignored)

Press the [HOME] key to show or hide the menu

Use the [↑][↓] arrow keys to switch between menu items

Use the [←][→] arrow keys to modify the content of the selected item

Press [PgDn] to enter a sub-menu

Press [Backspace] to return to the previous menu
